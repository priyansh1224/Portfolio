# Priyanshu Oli - Portfolio Website

A modern, responsive portfolio website showcasing my web development journey and projects.

## Features
- Responsive design for all devices
- Mobile-friendly hamburger menu
- Smooth animations with AOS
- Interactive project cards
- Contact form
- Resume download

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Lottie Animations
- AOS (Animate On Scroll)

## Setup
1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!
